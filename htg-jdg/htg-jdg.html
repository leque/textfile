  • 使用フォント: Unicode変体仮名フォント
  • 変体仮名一覧と字源は Unicode 13.0 時点の Code Charts Kana Supplement と Kana Extended-A によった。
  • 甲乙の別は三省堂『時代別国語大辞典 上代編』の付録「主要万葉仮名一覧表」に従った。
    • 上記に記載なきものは甲乙を「-」とした。
    • 「も」の変体仮名で古事記に用例のないものは甲乙を「両」とした。
    • 「え」の変体仮名に関しては便宜上ア行エを甲、ヤ行エを乙とした
  • 「め」の仮名には字源がメ乙類の万葉仮名として使われたものがない。
  • 「の」の仮名には字源がノ甲類の万葉仮名として使われたものがない。
    • 「ぬ」の変体仮名の字源「努（𛂏）」はヌ・ノ甲、「奴（𛂐）」「怒（𛂑）」はド甲・ヌ・ノ甲での用例がある。
